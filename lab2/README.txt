READ ME FILE
This code was made by Andrew Koski. Code is made to read inputs from the console/user. 
Reads total dice, the number of sides, amount of players, player names, and which game they wish to play(knockout or Boston dice.) 
The code takes the input and runs the predetermined for loops to give scores to all players
After a score is obtained for a player, it then places the number into the Sum.txt file the code the finds the winner of the game from the Score.txt file.

How to start:
Open code in replit.com in a C++ file to add the code to and click the run button above to start the code. When prompted by the program, input all asked values.
